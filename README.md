# ETHOS-FINE-DH-Cascade
District Heating Cascade Framework extending ETHOS.FINE with sustainability, resilience, flexibility, and policy assessment capabilities.

ETHOS-FINE-DH-Cascade is a district heating sustainability assessment framework developed within the PRG2701 project and built upon the ETHOS.FINE energy system modelling framework.

The framework extends the original ETHOS.FINE multi-regional workflow with district heating cascade modelling, sustainability assessment, resilience analysis, flexibility indicators, policy compliance evaluation, and multi-criteria decision support.

## Key Features

* District heating cascade modelling
* National-scale district heating planning
* Sustainability assessment framework
* Resilience analysis
* Flexibility indicators
* Policy compliance evaluation
* TOPSIS-based decision support
* Government reporting workflow

## Scientific Background

The framework has been developed as part of the PRG2701 Next-Generation District Heating project.

The work builds upon the ETHOS.FINE framework developed by Forschungszentrum Jülich and extends it with district-heating-specific sustainability and decision-support capabilities.
